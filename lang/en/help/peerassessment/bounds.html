Lower Bound

This specifies the value that below which the system will highlight a student's rating.

Given the Peer Assessment Tool uses a Likert Scale of 1-5 values smaller than 1 are not sensible!

Upper Bound

Given the Peer Assessment Tool uses a Likert Scale of 1-5 values greater than 5 are not sensible!
